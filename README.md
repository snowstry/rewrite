### Setup

**Dependencies**

-   [Baker](https://github.com/rv178/baker) optional, unless you wanna run commands yourself.
-   Yarn (for tailwindcss)
-   Rust
-   Trunk
-   Wasm target `wasm32-unknown-unknown`

```bash
bake setup
```

### Running

```bash
bake
```
